## Practical 1: Guessing Game
A console-based game where the user attempts to guess a randomly generated number between 1 and 100. The program provides hints (higher or lower) and tracks a score based on the number of attempts.

## Practical 2: Rock, Paper, Scissor Game
A classic game against the computer. The user inputs their choice, and the computer randomly selects one of the three options to determine a winner, loser, or a tie.
